# JAY OS — Personal Command Center

A premium, dark-mode personal operating system that helps organize and execute
life goals across fitness, music, productivity, planning, and personal growth —
all from a single dashboard.

Built as a fast, lightweight, zero-dependency web app. No build step, no backend —
just open `index.html`. All data is stored locally in your browser (`localStorage`).

## Features

- **Dashboard** — today's priorities, daily checklist, weekly focus, motivational
  quotes, recent activity, and quick access to every section.
- **Goals** — Daily, Weekly, Monthly and Yearly goal cards with priorities, due
  dates, notes, edit and complete.
- **Weekly Planner** — drag-and-drop tasks between days, time blocking, priority
  indicators, weekly focus and review.
- **Fitness** — 148 → 180 lbs tracker, 3-day training split, workout library,
  set-by-set workout tracker, and exercise-mastery roadmaps (muscle-up, handstand,
  225 bench, 315 squat, pull-ups, dips).
- **Diet** — weight-gain meal plans with macros, categorized grocery list, and a
  cheap-bulking section.
- **Music** — beat sales, social media services and website development goals
  with a $500/month income target ring.
- **Vision Board** — image + note cards across Financial, Fitness, Music,
  Lifestyle and Personal Development.
- **AI Coach** — a built-in mentor/strategist that analyzes your real data and
  gives next actions, motivation and habit feedback.
- **Reminders** — daily, weekly and one-time reminders with browser notifications.
- **Settings** — theme, profile, notifications, plus export / import / reset of
  all your data.

Also includes global search, mobile responsiveness, a quick-add button, and
smooth animations throughout.

## Tech

- Vanilla HTML, CSS and JavaScript — no frameworks, no dependencies.
- `index.html` · `styles.css` · `app.js`

## Run it

Open `index.html` in any modern browser, or serve the folder:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Design

- Palette: black primary, gold secondary, white accent.
- Dark mode by default, with a light theme available in Settings.
- Inspired by Apple, Notion, Linear and Arc.
